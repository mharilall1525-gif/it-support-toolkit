# Windows IT Support Toolkit (PowerShell)

## Overview
This project is a PowerShell-based IT support toolkit designed to help diagnose and troubleshoot common issues on Windows systems.

It automates tasks that are typically performed in IT support roles, such as checking system performance, reviewing error logs, and testing network connectivity.

---

## Features
- Full system diagnostics (CPU, RAM, disk usage)
- Top processes by CPU usage
- Network checks (IP configuration + ping test)
- Event log analysis (recent system errors)
- Temporary file cleanup
- Exportable system report (report.txt)

---

## Technologies Used
- PowerShell
- Windows system utilities (ipconfig, event logs, etc.)
